## 一、装置是怎么工作的？

```
                        ┌──────────────────┐
                       │      浮子         │  ← 漂在海面上，随波浪上下动
     波浪 →            │   (圆柱+圆锥壳)    │
                        │                  │
                        │   ┌──────────┐   │
                        │   │   振子    │   │  ← 在浮子内部，沿中轴上下滑动
                        │   │ (质量块)  │   │
                        │   ├──────────┤   │
                        │   │ 弹簧 阻尼 │   │  ← PTO 系统 = 弹簧 + 阻尼器
                        │   ├──────────┤   │
                        │   │  中轴底座 │   │  ← 固定在浮子上
                        └───┴──────────┴───┘
       ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~  海面
```

**工作原理（三句话版）**：

1. 波浪推着浮子上下晃（**垂荡运动**）
2. 浮子内部的振子因为惯性"跟不上"浮子的运动，两者出现**相对运动**
3. 相对运动驱动阻尼器（就像活塞一样）做功 → **发电**

**核心思维**：这不是一个物体，而是**两个物体互相耦合**的振动系统。要分别对浮子和振子列方程。

### 2.3 需要自己计算的参数：静水恢复力系数

> K_静水 **没有直接给出**，需要自己算。

**物理含义**：浮子偏离平衡位置时，浮力变化会产生一个"拉回来"的力，与位移成正比。
比例系数就是 K_静水，相当于水给浮子提供的等效"弹簧刚度"。

**计算公式**：

```
K_静水 = 海水密度 × 重力加速度 × 水线面积
       = ρ · g · A_水线
       = ρ · g · π · r²
```

**代入数值**：

```
K_静水 = 1025 × 9.8 × π × (1)²
       = 1025 × 9.8 × 3.1416
       ≈ 31559.7  N/m
```

> 水线面积 A_水线 = πr² 是因为浮子圆柱部分半径为 1m，水面截出的截面是一个半径为 1m 的圆。

## 三、核心物理概念（用人话讲）

### 概念 1：垂荡位移 z

```
      ↑ 上方（负方向）
      |
  ────┼──── 平衡位置（z = 0）
      |
      ↓ 下方（正方向）
```

- **z_b(t)**：浮子在时刻 t 偏离平衡位置的距离，向下为正
- **z_z(t)**：振子在时刻 t 偏离平衡位置的距离，向下为正
- **ż_b(t)**：浮子的速度（位移对时间的导数）
- **z̈_b(t)**：浮子的加速度（速度对时间的导数）

### 概念 2：附加质量 m_add

> 你在水里用手推一个气球，感觉比在空气中重很多。
> 因为你不光推动了气球，还推动了气球周围**跟着一起动**的水。

附加质量就是"被带动的那部分水"等效出来的质量。在方程里，它**加在浮子质量上**：

```
有效质量 = m_b + m_add = 4866 + 1335.535 = 6201.535 kg
```

### 概念 3：兴波阻尼 C_波

> 你在水面上下拍手，水面会泛起一圈圈波纹向外扩散。
> 这些波纹带走了能量，对你的手形成了阻力。

浮子在水面上下晃时也会向四周辐射波浪，受到的阻力就是兴波阻尼力：

```
F_兴波 = −C_波 · ż_b
```

注意负号：**阻力的方向和运动方向相反**。比如浮子向下运动（ż_b > 0），兴波阻尼力朝上（负）。

### 概念 4：静水恢复力 K_静水

> 把皮球按进水里再松手，它自己弹回来。

浮子偏离平衡位置后，浸在水里的体积变了 → 浮力变了 → 产生一个"推回去"的力：

```
F_静水 = −K_静水 · z_b
```

- 浮子向下走了 z_b（下沉）→ 排开水变多 → 浮力变大 → 向上的力 → 负号
- 浮子向上走了 z_b（上浮）→ 排开水变少 → 浮力变小 → 向下的净力 → 也是负号（因为此时 z_b < 0）

**它就像一个弹簧，总是把浮子往平衡位置拉。**

### 概念 5：波浪激励力 f·cos(ωt)

> 站在海边，一个浪打过来，你被推了一下。

波浪对浮子施加一个周期性的外力：

```
F_激励(t) = f · cos(ωt)
```

- f = 6250 N 是力的最大值（振幅）
- ω = 1.4005 s⁻¹ 决定了力变化的快慢
- 力的大小随时间按余弦规律变化：有时最大（cos = 1），有时为零（cos = 0），有时反向（cos < 0）

### 概念 6：PTO 系统（弹簧 + 阻尼器）

- **弹簧**：提供与相对位移成正比的力

```
F_弹簧 = k · (z_b − z_z)
```

如果浮子比振子更往下（z_b > z_z），说明底座跟着浮子下移而振子没跟上，弹簧被**拉伸**，弹簧把两端往中间拉 → 推浮子往上↑、推振子往下↓。

- **阻尼器**：提供与相对速度成正比的力（情况 1）

```
F_阻尼 = c · (ż_b − ż_z)
```

如果浮子比振子动得更快（ż_b > ż_z），阻尼器给浮子一个向上的阻力、给振子一个向下的推力。

> 浮子和振子是**靠 PTO 系统连接**的，它们之间的力是**作用力和反作用力**关系：
> PTO 给浮子的力 = −(PTO 给振子的力)

## 四、受力分析——一步步推出方程

### 4.1 约定

- 向下为**正方向**
- 所有位移从静水平衡位置开始算
- 初始条件：t = 0 时，z_b = ż_b = z_z = ż_z = 0

### 4.2 浮子的受力（5 个力）

| #  | 力的名称   | 方向         | 公式                           | 解释                             |
| -- | ---------- | ------------ | ------------------------------ | -------------------------------- |
| ① | 波浪激励力 | ↓（正）     | f·cos(ωt)                    | 波浪推浮子，周期性变化           |
| ② | 附加惯性力 | 与加速度反向 | −m_add·z̈_b                 | 带动水一起加速，等效于"质量变大" |
| ③ | 兴波阻尼力 | 与速度反向   | −C_波·ż_b                   | 浮子辐射波浪的能量损失           |
| ④ | 静水恢复力 | 指向平衡位置 | −K_静水·z_b                  | 浮力变化产生，像弹簧一样         |
| ⑤ | PTO 作用力 | —           | −k(z_b−z_z) − c(ż_b−ż_z) | 弹簧 + 阻尼器对浮子的反作用力    |

### 4.3 振子的受力（2 个力）

| #  | 力的名称   | 方向 | 公式           | 解释                                 |
| -- | ---------- | ---- | -------------- | ------------------------------------ |
| ⑥ | PTO 弹簧力 | —   | +k(z_b−z_z)   | 弹簧推振子（和推浮子的力方向相反）   |
| ⑦ | PTO 阻尼力 | —   | +c(ż_b−ż_z) | 阻尼器推振子（和推浮子的力方向相反） |

> 注意 ⑤ 和 ⑥⑦ 的符号关系：**浮子和振子受到的 PTO 力总是大小相等、方向相反**。这是牛顿第三定律。

### 4.4 应用牛顿第二定律

> **牛顿第二定律**：质量 × 加速度 = 所有外力的总和

#### 浮子方程

```
浮子质量 × 浮子加速度 = ① + ② + ③ + ④ + ⑤

m_b · z̈_b = f·cos(ωt) − m_add·z̈_b − C_波·ż_b − K_静水·z_b − k(z_b−z_z) − c(ż_b−ż_z)
```

把含有 z̈_b 的项全部移到等号左边：

```
(m_b + m_add)·z̈_b + C_波·ż_b + K_静水·z_b + k(z_b−z_z) + c(ż_b−ż_z) = f·cos(ωt)
```

> 📌 **解读**：等号左边是"阻碍运动"的各类力，等号右边是"驱动运动"的波浪力。
> 这个方程描述了**浮子在各种因素的综合影响下如何运动**。

---

#### 振子方程

```
振子质量 × 振子加速度 = ⑥ + ⑦

m_z · z̈_z = k(z_b−z_z) + c(ż_b−ż_z)
```

移项整理：

```
m_z·z̈_z − k(z_b−z_z) − c(ż_b−ż_z) = 0
```

> 📌 **解读**：振子没有被波浪直接推，它只通过弹簧和阻尼器感受浮子的运动。
> 这个方程描述了**振子如何被浮子"带着"运动**。

## 五、完整数学模型（方程组）

将两个方程合并，得到**二自由度耦合受迫振动系统**：

```
╔═══════════════════════════════════════════════════════════════╗
║                                                               ║
║    (m_b + m_add)·z̈_b + C_波·ż_b + K_静水·z_b                 ║
║       + k(z_b − z_z) + c(ż_b − ż_z) = f·cos(ωt)              ║
║                                                               ║
║    m_z·z̈_z − k(z_b − z_z) − c(ż_b − ż_z) = 0                  ║
║                                                               ║
║    初始条件: z_b(0)=0, ż_b(0)=0, z_z(0)=0, ż_z(0)=0          ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

## 六、怎么求解？（降阶法 + 数值解）

### 6.1 为什么不能手算？

方程组含有两个未知函数 z_b(t) 和 z_z(t) 及其导数，是**耦合的二阶常微分方程组**。虽然它有解析解（用复数表示法），但对于初学者，**数值求解更简单直观**，也能直接用于后续的各种复杂情形（非线性阻尼、多自由度等）。

### 6.2 降阶：把 2 个二阶方程变成 4 个一阶方程

令四个**状态变量**：

```
x₁ = z_b      （浮子位移）
x₂ = ż_b      （浮子速度）
x₃ = z_z      （振子位移）
x₄ = ż_z      （振子速度）
```

则原方程组的 z̈_b 和 z̈_z 可以写为：

```
z̈_b = [ f·cos(ωt) − C_波·x₂ − K_静水·x₁ − k(x₁−x₃) − c(x₂−x₄) ] / (m_b + m_add)

z̈_z = [ k(x₁−x₃) + c(x₂−x₄) ] / m_z
```

于是得到 4 个一阶方程：

```
ẋ₁ = x₂                                     (1)  位移的导数 = 速度

ẋ₂ = [f·cos(ωt) − C_波·x₂ − K_静水·x₁       (2)  速度的导数 = 加速度
       − k(x₁−x₃) − c(x₂−x₄)] / (m_b + m_add)

ẋ₃ = x₄                                     (3)  位移的导数 = 速度

ẋ₄ = [k(x₁−x₃) + c(x₂−x₄)] / m_z             (4)  速度的导数 = 加速度
```

加上初始条件：

```
x₁(0) = 0,  x₂(0) = 0,  x₃(0) = 0,  x₄(0) = 0
```

> 这就是标准形式 **ẋ = f(t, x)**，
> 可以直接丢给 Python 的 `scipy.integrate.solve_ivp` 去解。

```
单个波浪周期  T = 2π / ω = 2π / 1.4005 ≈ 4.487 秒

求解 40 个周期  t_max = 40 × 4.487 ≈ 179.5 秒
```

## 七、两种阻尼情况的处理

### 情况 1：线性阻尼（阻尼力与相对速度成正比）

```
F_阻尼 = c · (ż_b − ż_z) = c · v_rel
```

其中 c = 10000 N·s/m 是**常数**。这是最常规的阻尼模型，直接代入方程即可。

---

### 情况 2：非线性阻尼（阻尼力与相对速度的 0.5 次方成正比）

```
F_阻尼 = c · |ż_b − ż_z|^α · sign(ż_b − ż_z) = c · |v_rel|^0.5 · sign(v_rel)
```

其中 c = 10000，α = 0.5（幂指数）。

> **为什么加 sign(v_rel)**：因为 0.5 次方开出来总是正数，需要用符号函数来保留方向。
>
> 例如：
>
> - v_rel = 4 → |4|^0.5 × sign(4) = 2 × (+1) = +2
> - v_rel = −4 → |−4|^0.5 × sign(−4) = 2 × (−1) = −2

⚠️ 非线性阻尼方程不能再用简单的解析方法求解，但数值方法不受影响——只需要改一行代码。

## 八、Python 代码框架

### 8.1 完整可运行代码（情况 1 + 情况 2）

```python
import numpy as np
from scipy.integrate import solve_ivp
import pandas as pd

# ============================================================
# 参数定义（全部使用标准单位：kg, m, s, N）
# ============================================================

# --- 浮子参数 ---
m_b = 4866.0            # 浮子质量 (kg)
r = 1.0                 # 浮子底面半径 (m)

# --- 振子参数 ---
m_z = 2433.0            # 振子质量 (kg)

# --- 环境参数 ---
rho = 1025.0            # 海水密度 (kg/m³)
g = 9.8                 # 重力加速度 (m/s²)

# --- PTO 参数 ---
k_spring = 80000.0      # 弹簧刚度 (N/m)

# --- 水动力参数（附件3，问题1行，只取垂荡列）---
omega = 1.4005          # 波浪圆频率 (s⁻¹)
m_add = 1335.535        # 垂荡附加质量 (kg)
C_wave = 656.3616       # 垂荡兴波阻尼系数 (N·s/m)
f_amp = 6250.0          # 垂荡激励力振幅 (N)

# --- 计算静水恢复力系数 ---
K_hydro = rho * g * np.pi * r**2   # ≈ 31559.7 N/m

print(f"静水恢复力系数 K_hydro = {K_hydro:.2f} N/m")

# ============================================================
# ODE 右端函数
# ============================================================

def ode_system(t, x, c_damp, nonlinear=False, alpha=0.5):
    """
    计算状态变量的导数 ẋ = f(t, x)

    参数:
        t: 当前时间
        x: 状态向量 [z_b, ż_b, z_z, ż_z]
        c_damp: 阻尼系数
        nonlinear: True=非线性阻尼(情况2), False=线性阻尼(情况1)
        alpha: 非线性阻尼的幂指数(情况2用)
    """
    z_b, zb_dot, z_z, zz_dot = x

    # 相对位移
    rel_disp = z_b - z_z

    # 相对速度
    rel_vel = zb_dot - zz_dot

    # 阻尼力（线性 vs 非线性）
    if nonlinear:
        F_damp = c_damp * np.abs(rel_vel) ** alpha * np.sign(rel_vel)
    else:
        F_damp = c_damp * rel_vel

    # PTO 总力 = 弹簧力 + 阻尼力
    F_PTO = k_spring * rel_disp + F_damp

    # === 浮子加速度 ===
    # 方程: (m_b + m_add)·z̈_b = f·cos(ωt) − C_波·ż_b − K_静水·z_b − F_PTO
    F_excite = f_amp * np.cos(omega * t)          # 波浪激励力
    F_total_buoy = F_excite - C_wave * zb_dot - K_hydro * z_b - F_PTO
    zb_ddot = F_total_buoy / (m_b + m_add)

    # === 振子加速度 ===
    # 方程: m_z·z̈_z = F_PTO
    zz_ddot = F_PTO / m_z

    return [zb_dot, zb_ddot, zz_dot, zz_ddot]


# ============================================================
# 求解函数
# ============================================================

def solve_problem(c_damp, nonlinear=False, alpha=0.5):
    """
    求解 40 个周期的运动，返回结果 DataFrame。

    参数:
        c_damp: 阻尼系数
        nonlinear: 是否使用非线性阻尼
        alpha: 非线性阻尼的幂指数
    """
    T_period = 2 * np.pi / omega          # 单周期 ≈ 4.487 s
    t_max = 40 * T_period                 # 最大时间 ≈ 179.5 s
    t_eval = np.arange(0, t_max, 0.2)     # 步长 0.2 s

    # 初始条件 [z_b, ż_b, z_z, ż_z] = [0, 0, 0, 0]
    x0 = [0.0, 0.0, 0.0, 0.0]

    # 调用 ODE 求解器
    sol = solve_ivp(
        ode_system,
        t_span=(0, t_max),
        y0=x0,
        t_eval=t_eval,
        method='RK45',
        args=(c_damp, nonlinear, alpha),
        rtol=1e-8,
        atol=1e-10
    )

    # 整理结果
    df = pd.DataFrame({
        '时间 t (s)': sol.t,
        '浮子位移 z_b (m)': sol.y[0],
        '浮子速度 ż_b (m/s)': sol.y[1],
        '振子位移 z_z (m)': sol.y[2],
        '振子速度 ż_z (m/s)': sol.y[3]
    })

    # 计算相对速度（用于验证）
    df['相对速度 v_rel (m/s)'] = df['浮子速度 ż_b (m/s)'] - df['振子速度 ż_z (m/s)']

    return df


# ============================================================
# 主程序
# ============================================================

if __name__ == '__main__':
    # --- 情况 1：线性阻尼，c = 10000 ---
    print("正在计算 情况1（线性阻尼）...")
    df1 = solve_problem(c_damp=10000.0, nonlinear=False)
    df1.to_excel('result1-1.xlsx', index=False)
    print("  结果已保存到 result1-1.xlsx")

    # --- 情况 2：非线性阻尼，c = 10000, α = 0.5 ---
    print("正在计算 情况2（非线性阻尼）...")

    # --- 输出特定时刻的结果（论文中需要）---
    check_times = [10, 20, 40, 60, 100]

    for label, df in [("情况1", df1), ("情况2", df2)]:
```

## 九、后续问题：问题 2 ~ 问题 4 的扩展

这道题的四个问题是递进关系：

| 问题   | 自由度         | 阻尼类型      | 任务                                |
| ------ | -------------- | ------------- | ----------------------------------- |
| 问题 1 | 1（仅垂荡）    | 线性 / 非线性 | 正问题：已知 c 求运动               |
| 问题 2 | 1（仅垂荡）    | 线性 / 非线性 | 反问题：求最优 c 使功率最大         |
| 问题 3 | 2（垂荡+纵摇） | 线性          | 正问题：已知 c 和 c_转 求运动       |
| 问题 4 | 2（垂荡+纵摇） | 线性          | 反问题：求最优 c 和 c_转 使功率最大 |

**问题 1 是整个题目的基石**。把问题 1 的 ODE 求解搞懂后：

- 问题 2 只是在问题 1 基础上加一个**外层优化循环**（搜索最优阻尼系数 c）
- 问题 3 是从 4 个一阶方程扩展到 **8 个一阶方程**（增加纵摇的 2 个位移+速度）
- 问题 4 是问题 3 的基础上加**外层优化循环**（搜索两个最优阻尼系数）
